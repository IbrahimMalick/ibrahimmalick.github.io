Course Overview

Welcome to our comprehensive interactive course on Artificial Intelligence fundamentals. This course covers key concepts in search algorithms, propositional logic, and machine learning through interactive lessons and visualizations.

Module 1: Search and Problem Solving

Lesson 1.1: Formulating Search Problems

  • State Space: All possible configurations in our problem domain
  • Initial State: Our starting point
  • Goal State(s): The desired outcomes we're searching for
  • Actions: Operations that transform one state into another
  • Transition Model: Defines how actions change states
  • Action Cost Function: Assigns a numeric cost to each action

Lesson 1.2: Search Algorithms

Breadth-First Search (BFS)

  • Explores nodes level by level
  • Complete and optimal (uniform costs)
  • Memory-intensive for wide trees

Greedy Best-First Search (GBFS)

  • Uses a heuristic function prioritizing nodes
  • Often faster but not guaranteed to be optimal
  • Can get stuck in loops or dead ends

Lesson 1.3: Tree Search vs Graph Search

  • Tree Search:
    • Treats the search space as a tree
    • May revisit same states through different paths
    • Does not use memory to avoid redundant exploration
  • Graph Search:
    • Keeps track of visited states to avoid redundancy
    • More efficient for problems with cycles or multiple paths

Module 2: Propositional Logic

Lesson 2.1: Logical Agents

Logical agents apply logic to determine the consequences of known facts. They use knowledge representation and inference to make decisions.

Lesson 2.2: Syntax and Semantics

  • Symbols and Sentences: Building blocks of logical expressions
  • Truth Tables: Define how logical connectives behave
  • Logical Equivalence: Different sentences can represent the same truth

Lesson 2.3: Inference and Resolution

  • Modus Ponens: From "P → Q" and "P", infer "Q"
  • Resolution: A single inference rule for proving entailment
  • Forward and Backward Chaining: Reasoning methods for knowledge-based systems

Module 3: Introduction to Machine Learning

Lesson 3.1: What is Machine Learning?

Machine Learning is the study of algorithms that learn from and make predictions on data. It's a core subset of AI.

Lesson 3.2: Supervised Learning

  • Linear Regression: Predicts numeric outcomes
  • Logistic Regression: Classifies binary outcomes
  • Decision Trees: Model decisions via branching questions

Lesson 3.3: Unsupervised Learning

  • K-Means Clustering: Groups similar data points
  • Dimensionality Reduction: Simplifies data while preserving structure

Lesson 3.4: Neural Networks

Inspired by biological brains, neural networks learn patterns via layers of interconnected "neurons." Popular in deep learning.
